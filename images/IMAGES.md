# Project Images

This directory contains the figures and visual assets generated during model training and evaluation.

## Available Images

| Image | Description |
|--------|-------------|
| `confusion_matrix_linear_svm.png` | Confusion Matrix for the Linear Support Vector Machine |
| `confusion_matrix_rbf_svm.png` | Confusion Matrix for the Radial Basis Function (RBF) Support Vector Machine |
| `confusion_matrix_polynomial_svm.png` | Confusion Matrix for the Polynomial Support Vector Machine |
| `confusion_matrix_tuned_rbf_svm.png` | Confusion Matrix for the Tuned RBF Support Vector Machine |
| `baseline_model_comparison.png` | Performance comparison of the baseline SVM models |
| `cross_validation_comparison.png` | Performance comparison using 5-Fold Stratified Cross-Validation |
| `hyperparameter_tuning_comparison.png` | Comparison of the baseline and tuned RBF SVM models |
| `project_workflow.png` | Workflow diagram of the machine learning pipeline |

## Purpose

These images are generated from the Jupyter Notebook and are used throughout the project documentation, including the main `README.md`.

All figures are exported in PNG format to ensure high-quality rendering on GitHub.
