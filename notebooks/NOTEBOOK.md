# Project Notebook

This directory contains the Jupyter Notebook implementing the complete machine learning pipeline for credit card fraud detection using Support Vector Machines (SVMs).

---

## Notebook

**File Name**

`Credit_Card_Fraud_Detection.ipynb`

---

## Notebook Contents

The notebook includes the complete workflow of the project:

- Dataset Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning and Preprocessing
- Duplicate Removal
- Feature Scaling using StandardScaler
- Linear Support Vector Machine (SVM)
- Radial Basis Function (RBF) Support Vector Machine
- Polynomial Support Vector Machine
- 5-Fold Stratified Cross-Validation
- Hyperparameter Tuning using GridSearchCV
- Model Evaluation
- Performance Comparison
- Confusion Matrix Visualization

---

## Required Dataset

Download the dataset from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the file:

`creditcard.csv`

inside the project directory before executing the notebook.

---

## Execution Instructions

1. Install all required dependencies.

```bash
pip install -r requirements.txt
```

2. Launch Jupyter Notebook.

```bash
jupyter notebook
```

3. Open:

`Credit_Card_Fraud_Detection.ipynb`

4. Execute all cells sequentially from top to bottom.

---

## Expected Output

After successful execution, the notebook generates:

- Exploratory Data Analysis (EDA)
- Feature Correlation Analysis
- Baseline SVM Performance
- Cross-Validation Results
- Hyperparameter Tuning Results
- Confusion Matrices
- Final Performance Comparison
